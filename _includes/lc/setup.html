Git

El Git és un sistema de control de versions que permet controlar qui fa canvis a quin fitxer, i quan, i proporciona opcions per actualitzar i compartir una versió del codi fàcilment amb https://github.com.

Seguiu les instruccions de la lliçó per instal·lar Git en el vostre ordinador.

Necessitareu un compte de github.com per parts de la lliçó de Git. Comptes bàsics de GitHub són gratuits. Si us plau crea un compte a GitHub si encara no en tens un abans del taller. Considera quina informació personal vols compartir. Per exemple, consulta aquestes instruccions per mantenir la direcció de correu electrònic que entres a GitHub privada . Necessitaras un navegador compatible.
